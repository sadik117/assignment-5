Question-1: What are the different ways to select an element in the DOM?

There are some different ways to select elements in the DOM using JavaScript. I can select element by its class name, by its ID, by Tag Name, by Name attribute, by query selector. But by query selector and by Id name is used most.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

We use innerHTML when we need inserting or modifying HTML content dynamically. Again we use innerText when we have to only visible text, ignoring hidden elements. TextContent is used when we want all text, including hidden text, and to avoid parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a js systemic method to handle all childs with one function.

Question-4: What is event bubbling in the DOM?

Event bubbling is like a fish bubble mechanism. When we trigger on a target it bubbles from child to parents.

Question-5: How do you create, add, and remove elements using JavaScript?

To create a new HTML element we can use document.createElement(). To add an Element to the DOM we can use appendChild(). And to remove a specific child we can use removeChild() but for removing an element directly we can use remove().
